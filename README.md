# TechStock Manager

TechStock Manager е десктоп приложение за управление на складови наличности на техника. Приложението позволява добавяне, визуализиране, редактиране и изтриване на продукти чрез графичен интерфейс, разработен с Tkinter.

## Използвани технологии

- Python
- Tkinter
- SQLite
- decimal

## Основни функции

- Добавяне на продукт
- Визуализиране на всички продукти
- Редактиране на избран продукт
- Изтриване на избран продукт
- Проверка за празни полета
- Проверка за невалидни стойности
- Автоматично създаване на локална база данни `warehouse.db`

## Файлове в проекта

Основният файл на приложението е:

`techstock_manager.py`

При първо стартиране приложението автоматично създава локална база данни:

`warehouse.db`

В нея се съхранява таблицата `products`, която пази информацията за продуктите в склада.

## Инсталация

За стартиране на приложението е необходимо да има инсталиран Python 3.x.

Приложението използва стандартни Python библиотеки:

- `tkinter` – за графичния потребителски интерфейс
- `sqlite3` – за работа с локална SQLite база данни
- `decimal` – за проверка и обработка на ценови стойности

Не е необходима отделна инсталация на база данни. SQLite работи локално чрез файл.

## Стартиране чрез Command Prompt / Terminal

1. Отваря се папката, в която се намира файлът `techstock_manager.py`.
2. В адресната лента на папката може да се напише `cmd`, за да се отвори Command Prompt директно в тази директория.
3. Изпълнява се командата:

`python techstock_manager.py`

Ако командата `python` не работи, може да се използва:

`py techstock_manager.py`

Тази команда казва на Python интерпретатора да изпълни файла `techstock_manager.py`. След изпълнението ѝ се отваря графичният прозорец на приложението TechStock Manager.

## Стартиране чрез IDLE

1. Отваря се IDLE.
2. Избира се `File → Open`.
3. Зарежда се файлът `techstock_manager.py`.
4. Натиска се `Run → Run Module`.
5. Може да се използва и клавишът `F5`.
6. След стартиране се отваря прозорецът на приложението TechStock Manager.

## Стартиране чрез Spyder

1. Отваря се Anaconda Navigator.
2. Стартира се Spyder.
3. От менюто се избира `File → Open`.
4. Избира се файлът `techstock_manager.py`.
5. Натиска се бутонът `Run`.
6. Може да се използва и клавишът `F5`.
7. След стартиране се отваря графичният прозорец на приложението.

## Компилиране

Приложението е разработено на Python и не изисква предварително компилиране. То се стартира директно от `.py` файла чрез Python интерпретатора.

## Работа с приложението

След стартиране потребителят може да:

- въведе име на продукт;
- въведе количество;
- въведе цена;
- добави продукта чрез бутона „Добави продукт“;
- избере продукт от таблицата;
- редактира избрания продукт чрез бутона „Редактирай избран“;
- изтрие избрания продукт чрез бутона „Изтрий избран“.

При всяко добавяне, редактиране или изтриване таблицата се обновява автоматично.
