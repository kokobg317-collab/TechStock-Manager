# TechStock Manager

TechStock Manager е десктоп приложение за управление на складови наличности на техника.

## Използвани технологии

- Python
- Tkinter
- SQLite

## Основни функции

- Добавяне на продукт
- Визуализиране на продукти
- Редактиране на избран продукт
- Изтриване на избран продукт
- Автоматично създаване на локална база данни warehouse.db

# TechStock Manager

TechStock Manager е десктоп приложение за управление на складови наличности на техника. Приложението позволява добавяне, визуализиране, редактиране и изтриване на продукти чрез графичен интерфейс, разработен с Tkinter.

## Използвани технологии

- Python
- Tkinter
- SQLite
- decimal

## Основни функции

- Добавяне на продукт
- Визуализиране на всички продукти
- Редактиране на избран продукт
- Изтриване на избран продукт
- Проверка за празни полета
- Проверка за невалидни стойности
- Автоматично създаване на локална база данни `warehouse.db`

## Инсталация

За стартиране на приложението е необходимо да има инсталиран Python 3.x.

Приложението използва стандартни Python библиотеки:

- `tkinter` – за графичния интерфейс
- `sqlite3` – за работа с локална SQLite база данни
- `decimal` – за проверка и обработка на ценови стойности

Не е необходима отделна инсталация на база данни. SQLite работи локално чрез файл.
